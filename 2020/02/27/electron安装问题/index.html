electron安装问题

编程 electron 2020/02/27

electron的安装有个坑，会卡在

npm install -g cross-env

cross-env npm_config_electron_mirror=”https://npm.taobao.org/mirrors/electron/" npm_config_electron_custom_dir=”8.0.2” npm install

> 注意这里的8.0.2 是当前electron的最新稳定版本，根据需要替换，同时还要去查询淘宝源是否存在对应版本的镜像，否则会提示404
